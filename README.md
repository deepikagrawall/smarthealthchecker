# 🧠 Smart Health Extension

AI-powered wellness tracker for Chrome/Edge

## 📥 Installation

### One-Click Install (Recommended)
1. Download: [smart-health-extension.zip](https://github.com/yourusername/smart-health-extension/raw/main/smart-health-extension.zip)
2. Extract the folder
3. Go to `chrome://extensions/`
4. Enable **Developer Mode**
5. Click **Load unpacked**
6. Select the extracted folder

### Features
- 💧 Track water intake with smart reminders
- 🧍 Stand break notifications
- 🍱 Meal tracking
- 🤖 AI-powered insights
- 🏆 Streak tracking & achievements
- 📊 Analytics dashboard
